Branch Commands
       1. Creating a New Branch
              git checkout -b branch_name

       2. Switching Branches
              git checkout branch_name

       3. Merging Branches
              git checkout main
              git merge branch_name

       4. Deleting a Branch
              git branch -d branch_name

Repository Commands
       1. Cloning Repository
              git clone repo_url
       2. Repo Status
              git status
       3. Repository Log
              git log
       4. Remote Repository
              git remote -v

Commit Commands
       1. Staging Changes
              git add file_name           # For a Specific file
              git add .                   # For all files
       2. Committing changes
              git commit -m "message on commit"
       3. Checking All Commits
              git log
       4. Amend Last Commit
              git commit --amend --no-edit # Amend without changing message
       5. Revert & Unstage Changes
              git reset HEAD file_name     # Unstage specific file (No file given = Unstage all files)
              git checkout -- file_name    # Discard changes in working directory
